Editing & exporting

Templates are plain HTML + CSS. Edit the copy in each .html; restyle globally in ad.css. To regenerate the PNG exports at exact ad dimensions:

  • bash scripts/render-slicks.sh (headless Chrome), or
  • open a template in a browser and screenshot the canvas (square = 1080×1080, wide = 1200×628).

Both the buyer and operator editions share the same ad system. Add a new slick by copying any template and editing the copy.
