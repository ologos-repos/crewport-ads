# YouTube — Target-Market Copy

Paste-ready titles, descriptions, and short scripts for YouTube (and Shorts). Each block is copy-paste ready. Keep the link in the description's first line.

**Channel notes.** Titles carry the search/click weight (front-load the value, under ~60 chars). First line of the description shows above the fold — put the hook and `crewport.ai` there. Shorts: a strong first 2 seconds, one idea, end on the CTA. Confident, concrete, no hype. One audience per video.
